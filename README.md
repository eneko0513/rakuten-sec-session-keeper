# 楽天証券自動セッション維持

楽天証券のログインセッションを維持するChrome拡張機能です。  
指定した間隔ごとにページ内の楽天証券ロゴをクリックし、セッションタイムアウトを防ぎます。

## 機能

- 指定間隔（5/10/15/20分、テスト用10秒）ごとに自動でページ操作
- バッジに次回実行までのカウントダウンを表示
- 一時停止 / 再開
- 直近5件の実行履歴（成否）をポップアップに表示

## インストール

1. このリポジトリをクローンまたはZIPダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオン
4. 「パッケージ化されていない拡張機能を読み込む」→ このフォルダを選択

## 使い方

1. 楽天証券にログインしたタブを開いたままにする
2. 拡張機能アイコンのバッジに残り秒数が表示される
3. ポップアップからインターバルの変更・一時停止が可能

## 注意

- パスワードや認証情報は一切保存しません
- 保存されるのはインターバル設定値のみ（`chrome.storage.local`）
- 楽天証券のタブが開いていない場合、またはログアウト状態の場合は操作をスキップします
